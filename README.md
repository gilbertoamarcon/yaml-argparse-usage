# yaml-argparse-usage/

A simple usage example of yamlcpp and cxxopts.


## Dependencies

```
sudo apt-get -y install libyaml-cpp0.5v5
sudo apt-get -y install libyaml-cpp-dev
```


## Build

```
mkdir build
cd build
cmake ..
make
```

## Generate problems

```
../bin/main -i ../data/test.yaml -o ../data/test.yaml
```
